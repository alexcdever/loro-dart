LoroDoc class

Loro文档Dart包装类

这个类提供了对Rust Loro库的面向对象封装，用于管理CRDT文档。 它封装了底层的FFI调用，提供了友好的Dart API接口。

示例用法：

final doc = LoroDoc();
doc.insertText("Hello, World!", 0);
doc.commit();
final text = doc.getText();
print(text); // 输出: Hello, World!
doc.dispose();

注意：使用完文档后，必须调用dispose方法释放资源， 或者使用late final和addPostFrameCallback确保资源被正确释放。
