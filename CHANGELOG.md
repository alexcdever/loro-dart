# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.2] - 2025-12-17

### Fixed
- Fixed SDK constraint to support inline-class feature
- Added missing path dependency for tool scripts
- Fixed web platform binding generation issues

### Changed
- Updated pubspec.yaml with proper SDK constraints (>=3.3.0)
- Improved web platform support

## [1.0.0] - 2025-12-16

### Added
- Initial release of loro_dart
- Complete FFI bindings for Loro CRDT
- Support for all major platforms (Android, iOS, Windows, Linux, macOS)
- Text, List, Map, Tree, Counter, and MovableList containers
- Document import/export functionality
- Event subscription system
- Undo/Redo manager
- Comprehensive examples and documentation

### Features
- High-performance CRDT operations
- Real-time collaboration support
- Type-safe Dart API
- Automatic resource management
- Cross-platform compatibility

## [Unreleased]

### Planned
- Additional example applications
- Performance benchmarks
- Extended documentation
- Integration guides
